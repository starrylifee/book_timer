# 아침 독서 미션 보드

전자칠판용 아침 독서 미션 화면입니다.

## 주요 기능

- 16:9 전자칠판 한 화면 고정 레이아웃
- 학생 번호를 설정창에서 자유롭게 늘리거나 바꿀 수 있음
- 학생별 이름, 별명을 선택적으로 저장하고 화면 표시 방식 선택 가능
- 첫 터치 시작, 5분 뒤 재터치 성공, 일찍 오면 1분 추가
- 교사 일괄 승인 버튼으로 성공 학생 승인 처리
- 선택형 포인트 API 연동: GROWND 기본값 포함, `API URL`, `API 키`, `반 ID`, `점수` 설정 가능
- 날짜별 `localStorage` 저장
- 오늘 CSV, 전체 CSV 다운로드
- 설정창에서 학교명, 학급명, 기본 독서 시간, 추가 시간 저장
- 설정창에서 오늘의 알림, 중요사항, 요일별 시간표 저장

## 사용 방법

1. 브라우저에서 `index.html`을 엽니다.
2. 오른쪽 위 `설정`에서 학교명, 학급명, 독서 시간을 먼저 입력합니다.
3. 오늘 알림, 중요사항, 시간표를 입력합니다.
4. 학생 목록을 한 줄에 한 명씩 `번호`, `번호,이름`, `번호,이름,별명` 형식으로 입력합니다.
5. 필요하면 포인트 승인 연동에 `API URL`, `API 키`, `반 ID`, `점수`를 입력합니다. 기본값은 GROWND 기준으로 채워집니다.
6. 학생은 자기 번호를 터치해서 시작하고, 다시 와서 같은 번호를 터치해 성공 여부를 확인합니다.
7. 교사는 `성공 학생 일괄 승인` 버튼으로 성공 학생을 한 번에 승인할 수 있습니다.

## 포인트 연동 메모

- GROWND 기본 URL은 `https://growndcard.com/api/v1/classes/{classId}/students/{studentCode}/points` 이고, 인증 헤더 기본값은 `X-API-Key`입니다.
- 승인 시 학생 번호를 `studentCode`로 사용해 학생별 POST를 보내고, 본문은 `type`, `points`, `description`, `source`, `metadata` 형식으로 전송합니다.
- `metadata`에는 `date`, `approvedAt`, `studentCode`, `studentNumber`, `studentName`, `studentNickname`을 함께 담습니다.
